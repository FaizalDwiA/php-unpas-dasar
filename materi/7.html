Array

Array

  • Sebuah variable yang bisa menampung lebih dari 1 nilai
  • Element pada array boleh memiliki tipe data yang berbeda
  • Pasangan antara key & value
  • Key-nya adalah index, yang dimulai dari 0

Membuat Array

  • Cara lama

    <?php
    $hari = array("Senin", "Selasa", "Rabu");
?>

  • Cara Baru

    <?php
  $bulan = ["Januari", "Februari", "Maret"];
?>

  • Dengan Type Data Yang Berbeda

    <?php
    $arr1 = [123, "tulisan", false];
?>

Menampilkan Array

Jika menggunakan echo akan error

<?php
    $arr1 = [123, "tulisan", false];
    echo $arr1;
?>

Hasil :

Solusi : var_dump() / print_r()

<?php
    $arr1 = [123, "tulisan", false];
    var_dump($arr1);
?>

Hasil :

Menggunakan print_r & var_dump 

<?php
    $hari = array("Senin", "Selasa", "Rabu");
    $bulan = ["Jnauari", "Februari", "Maret"];
    var_dump($hari);
    echo "<br>";
    print_r($bulan);
?>

Menggunakan Foreach

<?php

?>
<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Latihan 2</title>
        <style>

                width: 50px;
                height: 50px;
                background-color: salmon;
                text-align: center;
                line-height: 50px;
                margin: 3px;
                float: left;
            }

            .clear {
                clear: both;
            }
        </style>
    </head>
    <body>
        <?php foreach( $angka as $a ) { ?>
        <div class="kotak"><?php echo $a; ?></div>
        <?php } ?>
    </body>
</html>
